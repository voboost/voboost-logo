# Voboost Logo

Official logo assets for the voboost project.

## Logo Variants

| File                   | Description                        |
|------------------------|------------------------------------|
| voboost-logo-plain.svg | Default logo, optimized for web    |
| voboost-logo-fire.svg  | Fire variant with gradient effects |
| voboost-logo-team.svg  | Team chat version                  |

Source files (with `_source` suffix) contain the original editable versions.
The `_grid` version contains grid/construction variant.

## Usage

### Allowed

- Reference in articles and blog posts about voboost
- Display when discussing the voboost project
- Include in presentations about voboost
- Link to this repository

### Not Allowed

- Commercial use without explicit permission
- Modifications or derivative works
- Use as your own brand or logo
- Implying endorsement by voboost
